# 고객 이탈 예측 프로젝트 요약

---

## 1. 프로젝트 개요

Telco 고객 데이터를 활용해 **이탈 예측 모델을 구축**하고, 이탈 전 위험 신호를 식별해 리텐션 전략으로 연결하는 것을 목표로 수행했다.

> BigQuery SQL 기반 전처리 → Colab 모델링 → SHAP 해석 → 비즈니스 전략 도출

---

## 2. 핵심 인사이트 (SQL 기반 1차 분석)

| 변수 | 주요 발견 |
|------|-----------|
| 계약 형태 | Month-to-Month 이탈률 **45.8%** vs Two Year **2.6%** |
| 인터넷 유형 | Fiber Optic 이탈률 **40.7%** — 가장 위험한 유형 |
| 가입기간 | 0–5개월 이탈률 **54.7%** → 길수록 급감 (48개월↑: 9.6%) |

## 3. 모델 성능 비교

| 모델 | F1-score | ROC-AUC | 비고 |
|------|----------|---------|------|
| Random Forest (전체) | 0.843 | 0.963 | 기준 모델 |
| XGBoost (전체) | 0.920 | 0.990 | 최고 성능, 누수 가능성 존재 |
| XGBoost v2 (누수 제거) | 0.692 | 0.893 | 보수적 검증 모델 |
| **XGBoost FS (선별 피처)** | **0.697** | **0.893** | **최종 운영형 모델** ✅ |
| XGBoost PCA | 0.658 | 0.875 | 차원 축소 시 성능 하락 |
| XGBoost + SMOTE | 0.662 | 0.880 | 오버샘플링 효과 없음 |

> XGBoost FS 선택 이유: 데이터 누수 없이 핵심 변수만으로 유의미한 성능 유지 + 실무 적용 가능

## 4. SHAP 분석 — 모델 해석

### 주요 변수 방향성

| 구분 | 피처 | 해석 |
|------|------|------|
| 🟢 보호 요인 | `Contract_Two Year` | 가장 강력한 이탈 억제 변수 |
| 🟢 보호 요인 | `Contract_One Year` | 장기 약정 → 이탈 위험 ↓ |
| 🟢 보호 요인 | `Dependents` | 부양가족 있을수록 안정 |
| 🟢 보호 요인 | `TenureInMonths` (↑) | 가입 기간 길수록 이탈 위험 ↓ |
| 🔴 위험 요인 | `MonthlyCharge` (↑) | 월 요금 높을수록 이탈 위험 ↑ |
| 🔴 위험 요인 | `TenureInMonths` (↓) | 가입 초기 이탈 위험 집중 |
| 🔴 위험 요인 | `InternetType_Fiber Optic` | Fiber Optic 고객의 위험 상대적으로 높음 |

## 5. 이탈 전 증상 — 고위험 조합

| Contract | TenureBand | InternetType | 실제 이탈률 |
|----------|-----------|--------------|------------|
| Month-to-Month | 0–5m | Fiber Optic | **74.0%** |
| Month-to-Month | 6–11m | Fiber Optic | **70.7%** |
| Month-to-Month | 0–5m | Cable | 64.3% |
| Month-to-Month | 0–5m | DSL | 61.4% |

> 대표 고위험 시나리오: **단기 계약 + 가입 초기 + Fiber Optic**
