# 📋 [최종보고서] Telco 고객 이탈 예측 및 사전 증상 분석
> **SQL 기반 데이터 엔지니어링부터 실무형 AI 모델 구축까지**

---

## 1. 프로젝트 요약 (Executive Summary)
* **목적:** 데이터 누수 없는 현실적인 이탈 예측 모델 구축 및 선제적 대응 전략 수립
* **도구:** BigQuery(데이터 정제), Colab(머신러닝), XGBoost(핵심 알고리즘)
* **성과:** 누수 변수 제거 후에도 **F1-score 0.697 / ROC-AUC 0.893**의 안정적 성능 확보

---

## 2. 데이터 엔지니어링 (BigQuery SQL)
데이터를 단순히 불러오는 데 그치지 않고, **분석 효율을 극대화**하기 위한 전처리를 수행했습니다.
* **통합 마트:** 6개 테이블을 고객 ID 기준으로 조인하여 `analysis_base` 생성
* **파생 변수:** `TenureBand`(가입구간), `ContractRisk`(계약위험도) 등 이탈 시그널 변수 10종 설계
* **초기 인사이트:** * **계약형태:** Month-to-Month 고객 이탈률 **45.8%** (최대 위험군)
    * **인터넷:** Fiber Optic 사용자의 높은 이탈률 확인

---

## 3. 모델링 실험 및 검증 (Model Pipeline)
정확도 수치보다 **"실제로 운영 가능한가?"**를 기준으로 4단계 실험을 거쳤습니다.

| 단계 | 모델명 | 특징 | 성능 (F1) | 결과 |
| :--- | :--- | :--- | :---: | :--- |
| **Step 1** | **XGBoost v1** | 전체 변수 사용 | **0.920** | **탈락** (데이터 누수 의심) |
| **Step 2** | **XGBoost v2** | 만족도 등 사후 변수 제거 | 0.692 | 통과 (현실적 예측력 증명) |

## 4. 이탈 전 증상 분석 (Risk Profiling)
모델이 예측한 고위험 고객들의 공통적인 특징(패턴)을 추출했습니다.

> **⚠️ 최우선 관리 대상 (TOP Risk Profile)**
> * **조건:** 월 단위 계약(`Month-to-Month`) + 가입 5개월 이내 + 광통신(`Fiber Optic`)
> * **분석 결과:** 해당 조합의 고객은 **평균 이탈 위험도 87.3%**를 기록함



---

## 5. 비즈니스 액션 플랜 (Strategic Suggestion)
분석 결과를 기반으로 즉시 실행 가능한 **4대 리텐션 전략**을 제안합니다.

1. **계약 전환:** 단기 계약자 대상 **장기 계약(1~2년) 전환 프로모션** 집중 투입
2. **초기 관리:** 가입 후 **0~6개월 고객 전담 온보딩 프로그램** 운영
3. **품질 케어:** 광통신(Fiber Optic) 고객 대상 **품질 만족도 정기 모니터링**
4. **타이밍:** 모델 예측 확률 기반 **1·3·6개월 시점별 자동 알림 및 혜택** 제공

---

## 6. 결론 및 시사점
* **기술적 신뢰:** 데이터 누수를 차단하여 실무 적용 시 성능 왜곡이 없는 모델을 완성함
* **운영 효율:** 복잡한 변수 없이 핵심 변수 조합만으로 고위험군을 정교하게 타겟팅 가능
* **기대 효과:** 이탈 징후가 뚜렷한 **초기/단기 계약 고객**에 마케팅 비용을 집중하여 리텐션 효율 극대화
